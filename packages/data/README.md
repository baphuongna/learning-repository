# Data Package

`packages/data` chua phan du lieu dung chung cua he thong:

- `prisma/schema.prisma`: schema nguon cho SQLite/Prisma
- `prisma/dev.db`: database local dung chung
- `prisma/seed.ts`: seed du lieu mau

Package nay khong phai backend runtime.
Backend runtime hien tai nam o `services/rust-doc-service`.
